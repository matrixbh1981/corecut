CoreCut API
REST API powering the CoreCut video editor. All endpoints are n8n webhooks — no API key required for public endpoints. Send JSON bodies for POST requests.
Base URL https://matrixbh.app.n8n.cloud/webhook
⚡
No Auth Required
All public endpoints work without API keys or tokens.
🌍
REST + JSON
Standard HTTP methods. Send Content-Type: application/json.
🤖
AI Powered
Whisper, OpenAI TTS, Replicate, and Shotstack integrated.
Authentication
CoreCut API currently uses a guest user model. Pass userId in every request body. For production, replace with your own JWT or session token system.

POST
/app-create-project
Create a new empty project. Returns a project ID used for all subsequent API calls.
Request Body
FieldTypeRequiredDescription
userIdstringrequiredUser identifier
titlestringrequiredProject name shown in the editor
Example Request
// POST /app-create-project
{
  "userId": "guest-user-001",
  "title": "My Vacation Video"
}
Response
201 Created 
{
  "success": true,
  "project_id": "proj-1784308806209-87298",
  "title": "My Vacation Video",
  "status": "draft"
}

POST
/app-upload
Upload a video or image file to Google Drive and attach it to a project. Send as multipart/form-data.
Form Fields
FieldTypeRequiredDescription
filebinaryrequiredVideo or image file
userIdstringrequiredUser identifier
titlestringrequiredFile/project title
projectIdstringoptionalAttach to existing project
Example Request
// JavaScript — multipart/form-data
const fd = new FormData();
fd.append('file', fileInput.files[0]);
fd.append('userId', 'guest-user-001');
fd.append('title', 'My Video');
fd.append('projectId', 'proj-...');

fetch('https://matrixbh.app.n8n.cloud/webhook/app-upload', {
  method: 'POST',
  body: fd
});
Response
201 Created 
{
  "success": true,
  "projectId": "proj-1784308806209-87298",
  "fileUrl": "https://drive.google.com/file/d/.../view"
}
POST
/app-ai-captions
Generate auto-captions from the project video using OpenAI Whisper. Returns 202 immediately — captions are saved to the project asynchronously.
Request Body
FieldTypeRequiredDescription
projectIdstringrequiredProject to transcribe
userIdstringrequiredUser identifier
languagestringoptionalen, ar, fr, es, de, ja, zh. Default: en
Example Request
{
  "projectId": "proj-1784308806209-87298",
  "userId": "guest-user-001",
  "language": "en"
}
Response
200 Accepted 
{ "status": "processing", "message": "Captions will be saved to project" }

POST
/app-render-video
Render a final MP4 video via Shotstack. Define clips, text overlays, audio, and output format. Rendering takes 30–90 seconds.
Request Body
FieldTypeRequiredDescription
projectIdstringrequiredTarget project
userIdstringrequiredUser identifier
editobjectrequiredShotstack edit object (clips, output)
Example Request
{
  "projectId": "proj-1784308806209-87298",
  "userId": "guest-user-001",
  "edit": {
    "clips": [
      {
        "asset": { "type": "video", "src": "https://drive.google.com/..." },
        "start": 0,
        "length": 15,
        "speed": 1
      },
      {
        "asset": { "type": "title", "text": "My Title", "style": "minimal" },
        "start": 0,
        "length": 3
      }
    ],
    "output": {
      "format": "mp4",
      "resolution": "hd",
      "aspectRatio": "9:16"
    }
  }
}
Response
200 OK 
{
  "success": true,
  "render_url": "https://cdn.shotstack.io/au/v1/.../output.mp4"
}
